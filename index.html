💡 Consejos generales

🥗 Alimentación

Prioriza alimentos con bajo índice glucémico. Evita azúcares refinados. Come en horarios regulares.

🚶 Actividad física

30 minutos diarios de actividad moderada. Caminar después de las comidas reduce los picos postprandiales.

💧 Hidratación

Bebe 1,5–2 litros de agua al día. Evita zumos y bebidas azucaradas.

😴 Descanso

Dormir 7–8 horas. El estrés y la falta de sueño elevan la glucemia.

🩺 Control médico

Revisiones periódicas. Revisa los pies a diario. Urgencias si glucosa >300 mg/dL.

⚙️ Configuración del Asistente IA

¿Por qué necesito una API Key?
El asistente usa la API de Claude (Anthropic) para analizar tus datos. Necesitas una clave propia gratuita para usarlo.

Cómo obtenerla:
1. Visita console.anthropic.com
2. Crea una cuenta gratuita
3. Ve a API Keys → Create Key
4. Copia la clave y pégala aquí
🔒 Tu clave se guarda solo en este dispositivo (localStorage). Nunca se envía a ningún servidor nuestro.
